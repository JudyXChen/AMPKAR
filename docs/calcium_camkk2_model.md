# Calcium-Dependent CaMKK2 Activation

## Motivation

The current model treats CaMKK2 as constitutively active at a fixed concentration. In reality, ionomycin has two parallel effects on AMPK:

1. **Metabolic arm** (currently modeled): Glycolysis inhibition → ATP depletion → AMP/ADP rise → nucleotide-dependent AMPK activation
2. **Calcium arm** (not modeled): Ca²⁺ influx → CaM activation → CaMKK2 activation → direct AMPK phosphorylation

We have RCamp data (`HeLaAMPKAR3_RCamp_Iono.npz`, `HeLaAMPKAR3_RCamp_2DG.npz`) that reports calcium levels at 0.5 min intervals, which can be used to constrain this pathway.

## Biological Pathway

### Step 1: Calcium-calmodulin binding

```
4 Ca²⁺ + CaM  ⇌  Ca₄CaM
```

Calmodulin (CaM) has 4 Ca²⁺-binding sites (EF-hand motifs), organized as two pairs in the N- and C-terminal lobes. Binding is cooperative within each lobe but independent between lobes. Reported Kd values range 3–20 μM per site; overall Hill coefficient for Ca²⁺-CaM is ~1.2–1.3 ([Crouch & Bhode, 1980](https://pubmed.ncbi.nlm.nih.gov/7407067/); [Stefan et al., 2008](https://pmc.ncbi.nlm.nih.gov/articles/PMC2435525/)).

### Step 2: CaMKK2 activation by Ca₄CaM (conformational relief of autoinhibition)

```
Ca₄CaM + CaMKK2_autoinhibited  ⇌  Ca₄CaM·CaMKK2*
```

**This is NOT a phosphorylation event.** CaMKK2 contains an autoinhibitory sequence (AIS) that maintains the kinase in an inactive conformation. Structural studies show the AIS does not block the substrate-binding pocket directly; instead it interacts with surfaces on the opposite side of the kinase domain. Ca₄CaM binding relieves autoinhibition by engaging the CaM-binding sequence, the N-lobe, RP-insert, and activation loop, assembling the catalytic spine into an active conformation ([Karamitros et al., 2022](https://pmc.ncbi.nlm.nih.gov/articles/PMC9570080/); [Bhattacharya et al., 2024](https://pmc.ncbi.nlm.nih.gov/articles/PMC11576191/)).

**Important: CaMKK2 has significant basal (autonomous) activity** — approximately 60–70% of total activity is Ca²⁺/CaM-independent. This autonomous activity is generated by autophosphorylation at Thr85, which creates a "molecular memory" maintaining CaMKK2 activation after transient Ca²⁺ signals. This distinguishes CaMKK2 from CaMKK1, which is strictly Ca²⁺/CaM-dependent ([Tokumitsu et al., 2011](https://pubmed.ncbi.nlm.nih.gov/21859090/); [O'Brien et al., 2015](https://www.nature.com/articles/srep14436)).

**Phosphorylation is mostly inhibitory for CaMKK2:**
- PKA phosphorylation at Ser495 impairs Ca²⁺/CaM-dependent activation
- CDK5/GSK3 phosphorylation reduces autonomous activity
- AMPK phosphorylation at Thr144 decreases basal activity (negative feedback)

([Karamitros et al., 2022](https://pmc.ncbi.nlm.nih.gov/articles/PMC9570080/); [Marcelo et al., 2016](https://www.nature.com/articles/srep43264))

### Step 3: AMPK phosphorylation (already in model)

```
Ca₄CaM·CaMKK2* + AMPK  →  pAMPK + Ca₄CaM·CaMKK2*
```

## Proposed Implementation: RCamp as Measured Input

Rather than modeling Ca²⁺ dynamics (which we directly observe via RCamp), use the RCamp signal as a time-dependent input and model only the Ca²⁺ → CaMKK2 activation mapping.

### Modeling considerations

Because CaMKK2 has ~60–70% autonomous activity without Ca²⁺/CaM, the activation function should have a nonzero baseline. We define the effective active CaMKK2 concentration as:

$$[\text{CaMKK2}^*](t) = [\text{CaMKK2}]_{\text{tot}} \left[ f_{\text{basal}} + (1 - f_{\text{basal}}) \cdot \frac{\text{Ca}(t)^n}{K_d^n + \text{Ca}(t)^n} \right]$$

where:

| Symbol | Definition | Units / Range |
|--------|-----------|---------------|
| $\text{Ca}(t)$ | Interpolated RCamp signal | Normalized $\Delta F/F$, range [0, 1] |
| $[\text{CaMKK2}]_{\text{tot}}$ | Total CaMKK2 concentration (existing `CaMKKtot`) | $\mu\text{M}$ |
| $f_{\text{basal}}$ | Fraction of autonomous activity without Ca²⁺/CaM | ~0.6–0.7 (fix or fit with tight prior) |
| $K_d$ | Half-maximal Ca²⁺ level for the CaM-dependent fraction | Normalized RCamp units (to fit) |
| $n$ | Hill coefficient | Dimensionless (to fit) |

**Limiting behavior:**

$$\text{Ca}(t) = 0 \implies [\text{CaMKK2}^*] = f_{\text{basal}} \cdot [\text{CaMKK2}]_{\text{tot}}$$

$$\text{Ca}(t) \gg K_d \implies [\text{CaMKK2}^*] \approx [\text{CaMKK2}]_{\text{tot}}$$

### Justification for lumped Hill function

The Hill function lumps two sequential binding events into a single phenomenological function of the observed RCamp signal:

$$\underbrace{4\,\text{Ca}^{2+} + \text{CaM} \rightleftharpoons \text{Ca}_4\text{CaM}}_{\text{Step 1: cooperative, } n_H \approx 1.2\text{–}1.3} \quad + \quad \underbrace{\text{CaMKK2} \rightleftharpoons \text{Ca}_4\text{CaM} \cdot \text{CaMKK2}^*}_{\text{Step 2: 1:1 binding, } n_H \approx 1}$$

This is justified because:
1. We observe Ca²⁺ directly (via RCamp), not Ca₄CaM
2. Ca₄CaM formation is fast relative to CaMKK2 conformational change
3. The Hill function captures the cooperative/sigmoidal response without mechanistic detail of each sub-step

### Hill coefficient interpretation

The effective $n$ reflects:

| Source | Contribution | Rationale |
|--------|-------------|-----------|
| Ca²⁺ → CaM | $n_H \approx 1.2$–$1.3$ | Moderate cooperativity within EF-hand pairs; lobes are independent |
| Ca₄CaM → CaMKK2 | $n_H \approx 1$ | Single CaM binds one CaMKK2 monomer |
| Sequential ultrasensitivity | May increase effective $n$ | Product of two Hill-like steps can sharpen response |

**Expected range: $n \approx 1$–$3$**

### Effect on existing model equations

#### MM nonessential model

Replace constant $[\text{CaMKK2}]_{\text{tot}}$ in CaMKK2-mediated phosphorylation fluxes:

**Current (constant CaMKK2):**

$$J_{\text{CaMKK},i} = \frac{k_{\text{phos}}^{\text{CaMKK}} \cdot [\text{CaMKK2}]_{\text{tot}} \cdot [\text{S}_i]}{K_m^{\text{CaMKK}} + [\text{S}_i]}, \quad \text{S}_i \in \{\text{AMPK},\; \text{AMP·AMPK},\; \text{ADP·AMPK},\; \text{ATP·AMPK}\}$$

**Proposed (calcium-dependent CaMKK2):**

$$J_{\text{CaMKK},i} = \frac{k_{\text{phos}}^{\text{CaMKK}} \cdot [\text{CaMKK2}^*](t) \cdot [\text{S}_i]}{K_m^{\text{CaMKK}} + [\text{S}_i]}$$

#### MA nonessential model

Replace constant $[\text{CaMKK2}]_{\text{tot}}$ in CaMKK2 binding fluxes (J7–J12 in `MA_nonessential_diffrax.py`):

**Current:**

$$J_{\text{bind},i} = k_{\text{on}}^{\text{CaMKK}} \cdot [\text{CaMKK2}]_{\text{tot}} \cdot [\text{S}_i] - k_{\text{off}}^{\text{CaMKK}} \cdot [\text{CaMKK2} \cdot \text{S}_i]$$

**Proposed:**

$$J_{\text{bind},i} = k_{\text{on}}^{\text{CaMKK}} \cdot [\text{CaMKK2}^*](t) \cdot [\text{S}_i] - k_{\text{off}}^{\text{CaMKK}} \cdot [\text{CaMKK2} \cdot \text{S}_i]$$

### New parameters

| Parameter | Symbol | Description | Prior expectation | Notes |
|-----------|--------|-------------|-------------------|-------|
| `f_basal_CaMKK` | $f_{\text{basal}}$ | Basal autonomous activity fraction | ~0.6–0.7 | Could fix at 0.65 or fit with tight prior |
| `Kd_CaCaMKK` | $K_d$ | Half-activation Ca²⁺ level (normalized RCamp units) | ~0.2–0.5 | Midrange of normalized RCamp signal |
| `n_CaCaMKK` | $n$ | Hill coefficient for lumped Ca²⁺→CaMKK2 activation | ~1–3 | Reflects cooperative Ca/CaM + CaMKK2 binding |

### Implementation steps

1. **Interpolate RCamp signal**: Use `diffrax.LinearInterpolation` to create `Ca(t)` from discrete RCamp measurements
2. **Modify ODE module**: Add `Ca(t)` interpolant as an attribute; compute `CaMKK2_active(t)` inside `__call__` using `t`
3. **Update `solve_traj()`**: Pass RCamp interpolant to the stressed-phase ODE (basal phase uses `Ca(t) = 0` → `CaMKK2_active = f_basal * CaMKK2_total`)
4. **Add priors**: `Kd_CaCaMKK`, `n_CaCaMKK`, and optionally `f_basal_CaMKK` as new free parameters
5. **Joint likelihood**: Fit to both AMPKAR and (optionally) RCamp data simultaneously

## Alternative: Full Calcium Cascade (More Complex)

Add explicit Ca²⁺, CaM, Ca₄CaM, CaMKK2, Ca₄CaM·CaMKK2* as ODE states and fit jointly to RCamp + AMPKAR. More mechanistic but requires many more parameters and modeling the calcium influx source. Not recommended unless the Hill function approach proves insufficient.
